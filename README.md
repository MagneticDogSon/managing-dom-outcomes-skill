## What is it?
This skill transforms an AI agent into a "Digital Assistant" (DA) that doesn't just respond to prompts but focuses on achieving specific user outcome by systematically bridging the gap between the current state and the desired result.

## Why use it?
While we often know *what* we want to achieve, we don't always have a clear algorithm to monitor progress. DOM provides a framework that:
1.  **Objectifies progress** via an A-F rating system.
2.  **Daemonizes the environment**: treats files, APIs, and processes as live data streams (Daemons).
3.  **Acts autonomously**: suggests or executes steps to improve the project's status.

## How to use

1.  **Installation**: Copy the contents of this repository to the `.agent/skills/managing-dom-outcomes/` folder in your project.
2.  **Activation**: Simply mention keywords in your chat with Antigravity: *DOM*, *Rating*, *Optimization*, or *Goal*.
3.  **Workflow**:
    - **Goals**: Define KPIs.
    - **Model**: Choose the achievement logic.
    - **Capture**: Gather data via built-in scripts.
    - **Rating**: Get a score from A (ideal) to F (critical).
    - **Action**: Execute recommendations to raise the rating.
    - **Adjustment**: Verify the outcome.

## Key Benefits
- **Proactivity**: The agent ensures the project stays on track before things go wrong.
- **Transparency**: You always know the "score" of your project (e.g., "Code Quality: C, Documentation: B").
- **Frictionless**: Minimal interruptions for the user ‚Äî the agent manages the "Daemon world" autonomously.
- **Scalability**: Works for everything from repository cleanup to complex business process management.
